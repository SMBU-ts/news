关于本站

「每日新闻」是一个基于 GitHub Pages 的纯静态新闻档案站点。所有内容按日期归档，结构清晰，便于长期维护与扩展。

技术特性

  • 多页结构：首页、归档页、关于页，以及每天的汇总页与新闻原文页，页面间通过统一导航互联。
  • 自动部署：每次向 main 分支提交，GitHub Actions 会自动构建并发布。
  • 自定义域名：通过仓库根目录的 CNAME 文件与 DNS 配置即可绑定自有域名。
  • SEO 基础：每页具备标题、描述、规范链接（canonical）、Open Graph、Twitter Card 与 JSON-LD 结构化数据，并附带 sitemap.xml 与 robots.txt。

如何新增一天的新闻

  1. 用 build_dashboard.py 生成当日报文，例如 ai-daily-2026-07-20.html。
  2. 将其放入 YYYY-MM-DD/<分类>/ 目录，例如 2026-07-20/ai-daily/。
  3. 运行 build_archive.py 重建首页、归档页、当日汇总页与站点地图。
  4. 提交并推送，站点会自动更新。
